# Healthcare-Analysis-Dashboard
Healthcare Analysis Dashboard – Power BI Project
📌Overview
This project analyses healthcare data to uncover key insights into patient trends, treatment costs, and hospital performance. The dashboard highlights important patterns in medical conditions, admissions, billing, insurance coverage, and patient outcomes, helping identify factors that impact healthcare efficiency and decision-making.
🛠Tools & Technologies Used
•	Power BI – Data visualization and dashboard creation 
•	Power Query – Data cleaning and transformation 
•	DAX – Calculated measures and KPIs
🎯Objective
•	Identify high revenue generating medical conditions 
•	Analyze patient admission patterns 
•	Evaluate insurance provider contribution 
•	Understand treatment costs and hospital stay duration
🔬 Methodology
•	Data Collection 
o	Imported healthcare dataset into Power BI. 
o	Dataset includes patient details, medical conditions, billing amount, insurance providers, admission type, test results, and hospital stay duration. 
•	Data Cleaning (Power Query) 
o	Checked and corrected data types (date, number, text). 
o	Handled missing or inconsistent values. 
o	Standardized fields like gender, admission type, and medical condition names. 
•	Data Transformation 
o	Created new columns such as hospital stay duration and admission year. 
o	Grouped data for analysis of billing totals, patient counts, and admission trends. 
•	Data Modeling (DAX Measures) 
o	Created key calculations: 
	Total Patients 
	Total Billing Amount 
	Average Billing Amount per Patient 
	Average Hospital Stay Duration 
	Percentage of Test Results 
•	Data Visualization 
o	Designed interactive dashboard using: 
	KPI Cards for key metrics 
	Bar charts for comparison 
	Donut charts for distribution 
	Line chart for yearly trends 
o	Added slicers for Year, Gender, and Medical Condition. 
•	Insight Generation 
o	Identified highest revenue generating medical condition. 
o	Analyzed billing contribution by insurance providers. 
o	Evaluated patient admission patterns and test result distribution. 
o	Determined average hospital stay duration. 
•	Dashboard Optimization 
o	Applied consistent color theme. 
o	Improved layout alignment and readability. 
o	Reduced unnecessary clutter for better user experience.
