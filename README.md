# 🏥 Healthcare Analysis Dashboard – Power BI Project

## 📌 Overview
This project analyzes healthcare data to uncover key insights into patient trends, treatment costs, and hospital performance. The dashboard highlights patterns in medical conditions, admissions, billing, insurance coverage, and patient outcomes, helping identify factors that impact healthcare efficiency and support better decision-making.

---

## 🛠 Tools & Technologies Used
- Power BI – Data visualization and dashboard creation
- Power Query – Data cleaning and transformation
- DAX – Calculated measures and KPIs

---

## 🎯 Objective
- Identify high revenue generating medical conditions
- Analyze patient admission patterns
- Evaluate insurance provider contribution
- Understand treatment costs and hospital stay duration

---

## 🔬 Methodology

### 1. Data Collection
- Imported healthcare dataset into Power BI
- Dataset includes patient details, medical conditions, billing amount, insurance providers, admission type, test results, and hospital stay duration

### 2. Data Cleaning (Power Query)
- Checked and corrected data types (date, number, text)
- Handled missing or inconsistent values
- Standardized fields such as gender, admission type, and medical condition names

### 3. Data Transformation
- Created new columns such as hospital stay duration and admission year
- Grouped data for analysis of billing totals, patient counts, and admission trends

### 4. Data Modeling (DAX Measures)
Created key calculations:
- Total Patients
- Total Billing Amount
- Average Billing Amount per Patient
- Average Hospital Stay Duration
- Percentage of Test Results

### 5. Data Visualization
Designed an interactive dashboard using:
- KPI Cards for key metrics
- Bar charts for comparison analysis
- Donut charts for distribution insights
- Line chart for yearly trend analysis

Added slicers for:
- Year
- Gender
- Medical Condition

### 6. Insight Generation
- Identified highest revenue generating medical condition
- Analyzed billing contribution by insurance providers
- Evaluated patient admission patterns and test result distribution
- Determined average hospital stay duration

### 7. Dashboard Optimization
- Applied consistent color theme
- Improved layout alignment and readability
- Reduced unnecessary clutter for better user experience
